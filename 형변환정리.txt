1) 형 변환 이란?

값을 다른 타입으로 변환 시키는걸 형변환이라 한다.


2) 업캐스팅이란 ? 다른말은 ?

작은 개념의 값을 큰 개념의 값으로 변환 하는것이며 자동 형변환 이라고도 한다.


3) 다운캐스팅이란? 다른말은?

큰 개념의 값을 강제로 낮추는 행위로 강제 형변환 이라고도 한다.


4) Car <- Truck 에서 부모/자식 클래스를 구분하시오

Car = > 부모 클래스 (Super Class)

Truck = > 자식 클래스 (Sub Class)


5) byte b = 100; int i = b; 내부에서 일어난 처리 내용을 쓰시오

byte 라는 작은 개념 (-127~128) 에서 int 라는 (-21억~21억) 큰 개념으로 변환돼 저장됐다 (자동형변환 업캐스팅)



6) int = i2 = 10; byte b2 = (byte) i2; 내부 처리내용을 쓰시오

int 라는 큰 개념에서 byte 라는 작은개념으로 강제 형변환이 돼서 저장했다 (다운캐스팅)
